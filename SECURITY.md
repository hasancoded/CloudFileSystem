## Reporting a Vulnerability

We take the security of CloudFileSystem seriously. If you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to `security@cloudfilesystem.example.com`.

You should expect to receive a response within 24 hours. If for some reason you do not, please follow up via email to ensure we received your original message.
